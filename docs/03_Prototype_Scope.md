# RoomLight Prototype Scope

The RoomLight prototype is a command-line Python application for hotel staff. It allows the user to create lighting scenes, assign them to room groups, and synchronize them across multiple rooms. The prototype is intended for hotel employees who need to manage room lighting settings without technical support. It proves that the core idea, “configure once, sync everywhere,” works practice and that the system can save time, support different room groups, and remain simple to use.

## Requirements addressed by the prototype

**001 Scenes can be synchronized automatically to all rooms**

This is the main requirement proved by the prototype. The demo shows how one scene can be created once and applied to multiple rooms automatically.

**003 The system has scenes for different situations**

The prototype proves this by allowing the user to create and manage different lighting scenes for different room situations.

**004 Hotel staff can adjust scenes without an electrician**

The prototype proves this by showing that hotel staff can modify scenes directly through the application without technical knowledge or installation work.

**008 The system is intuitive to use without instructions**

The prototype addresses this requirement by using a simple command-line workflow with clear actions and only a few steps.
